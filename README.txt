This is the readme documentation for this project. (2/9/2014)


To use, add a US stock symbol in the form on the top right.
To reset the information, click on the reset button.

This program is for the purpose of displaying how other stocks relate to each other.
To change the stock that all other stocks compare to, click on the appropriate square on the left bar.
The information displayed is from the dates January 1st to February 1st, 2014.
Information at each node corresponds to a different day among the month.

node server must be running before attemping to connect to the server!
